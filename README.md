# Binance ATR(14) Volatility Monitoring Tool
Live Market Monitoring and ATR(14) execution tool using Binance API.

This projects implements a Python based trading tool that monitors live cryptocurrency market data from Binance and 
calculates ATR(14).

## Features
- Live Binance Market Data
- ATR Volatility Calculation
- Execution Monitoring
- Python Trading Tool

## Technology Stack 
- Python
- Binance API
- Pandas
- Numbpy

## Use Case
This tool can be used by traders and quantitative researchers to monitor volatility conditions in cryptocurrency markets
assist with strategy execution based on ATR derived signals.
